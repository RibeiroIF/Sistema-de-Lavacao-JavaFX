<h1> Sistema de Lavação - JavaFX/Hibernate </h1>

<h3> Neste repositório apresentamos o <b>Sistema Norteador</b>, que consiste no <b>Sistema de Lavação</b> </h3>

<p> Este programa é uma aplicação Desktop voltada para uma empresa de lavação de carros, constituindo-se das seguintes funcionalidades:</p>
<ul>
  <li> Cadastros:
    <ul> 
      <li> Cor </li>
      <li> Marca </li>
      <li> Modelo </li>
      <li> Veículo </li>
      <li> Cliente </li>
      <li> Serviço </li>
    </ul>
  </li> 
  <li> Processos:
    <ul>
      <li> Ordem de serviço: </li>
    </ul>
  </li>
  <li> Gráficos:
    <ul>
      <li> Ordens Mensais </li>
      <li> Lucro Mensal </li>
    </ul>
  </li>  
  <li> Relatórios:
    <ul>
      <li> Serviços </li>
      <li> Ordens de Serviço (no menu de ordens) </li>
    </ul>
  </li>  
  <li> Controle:
    <ul>
      <li> Desconto por categoria </li>
      <li> Pontuação por serviço </li>
    </ul>
  </li>  
</ul>

<hr>

<p><b>Alterações a implementar no futuro:</b></p>
<ul>
  <li>Controle de exceções para exclusão de dados por associação</li>
  <li>Implementação do uso da Pontuação para pagamento dos serviços na ordem</li>

<hr>

<p> Para este projeto foram utilizadas as tecnologias e linguagens: </p>
<ul>
  <li> JavaFX (back-end) </li>
  <li> JasperSoft studio e Scenebuilder (front-end) </li>
  <li> Hibernate (framework) </li>
  <li> MySQL (banco de dados) </li>
</ul>
